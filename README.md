# soja-hisab
🧾 Soja Hisab (সহজ হিসাব) — A free, simple billing &amp; invoicing app built for Indian small shop owners. Create GST invoices, track customers, manage stock, send bills via WhatsApp, and monitor dues — all in one place. Supports English, Hindi &amp; Bengali. 100 free bills, then affordable subscription plans.
